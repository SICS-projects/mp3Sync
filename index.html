What is this project about:

You can imagine the mp3Sync project as a mp3 player for android. But this mp3 player does snycronization for you and as you wish to.

For example:

You have 3 devices(One Windows PC at home, one Windows PC at work and one Android mobile phone):

  • You add one song on the PC at home and you want to have that song automaticaly syncronized on the PC at work. This project does that for you.
  • You add one song on the PC at home and your want to have that song automatically syncronized with you Android mobile phone, as soon as it is connected to a WLan point. This project dies that for you.

This project should syncronize your songs at your command automaticaly if you wish to. Baybe you dont want a syncronization. Baybe you want some songs as stream if you press the play button.

What is the technical background behind the project:

The syncronization is desiged at folowing:

There is one server written in php. On this server is a mysql database. The server knows what mp3 file is on what device(if you have more than one device. for example: android mobile phone, and personal computer)

If some files where created new on one device, the server copies the files to the other device. But only you want that. If you dont want that, the server only regognizes that there are new files, that can be copied. If you want that files by another time, the server copies it.
